## Filters and Interactivity
Users can interact with the dashboard using filters for:
- **Policy Number**
- **Claim Number**
- **Customer ID**

These filters allow customized insights at policy, claim, or customer levels.

---

## Business Insights
- **High inactive policy rate** (58%) suggests opportunities for retention campaigns.  
- **Travel and Health policies** drive most premium revenue.  
- **Claim rejection rates** may require process review to improve customer satisfaction.  
- **Adults** file the most claims, indicating the importance of targeted coverage plans.

## Tools and Technologies
- **Data Source:** Insurance policy and claim data (synthetic)  
- **Tool Used:** Microsoft Power BI  
- **Techniques:** DAX calculations, Power Query data cleaning, data modeling, and interactive visual design.
